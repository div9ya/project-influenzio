# Influenzio

Influenzio is a modern web platform designed to connect **brands**, **freelancers**, and **influencers** for collaborations, campaigns, and professional networking.

This project is built with a clean frontend structure and backend integration to provide a smooth and scalable user experience.

---

## ✨ Features

- 🔐 User Authentication System
- 👤 Influencer & Freelancer Profiles
- 🏢 Brand Collaboration Platform
- 💬 Chat Application Integration
- 📱 Fully Responsive Design
- 🎨 Modern UI/UX
- ⚡ Fast & Lightweight
- 🌐 Flask-Based Web Application

## 🛠️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone <your-repository-link>
```

---

### 2️⃣ Navigate to the Project Directory

```bash
cd Influenzio
```

---

### 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

### 4️⃣ Activate Virtual Environment

#### Windows
```bash
venv\Scripts\activate
```

#### Mac/Linux
```bash
source venv/bin/activate
```

---

### 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 6️⃣ Run the Application

```bash
python app.py
```
